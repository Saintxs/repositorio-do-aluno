Seção 2: Ranking de Tecnologias

  1. SQL: Fundamental para a manipulação e consulta de bancos de dados relacionais estruturados.

  2. Python: Excelente para backend, automação de tarefas e análise de dados acadêmicos.

  3. Git: Essencial para o controle de versão de código e colaboração em projetos de software.

  4. CSS: Responsável por dar estilo, layout e identidade visual às aplicações construídas.

  5. HTML: A fundação absoluta de qualquer página web. Sem ele, a internet visual não existe.

Seção 3: Glossário de Termos Técnicos

HTML
Linguagem de marcação utilizada na construção de páginas web.
Padrão oficial mantido pelo W3C para estruturação de documentos hipertexto.
API
Conjunto de rotinas e padrões de programação para acesso a um aplicativo de software ou plataforma baseado na Web.
Mecanismo que permite que dois componentes de software se comuniquem usando um conjunto de definições e protocolos.
URL
Endereço Web
O link ou caminho padronizado utilizado para localizar e acessar um recurso específico na internet.
DOM
Document Object Model
A estrutura em árvore que representa o documento HTML, permitindo que linguagens como o JavaScript manipulem seus elementos.
CSS
Mecanismo para adicionar estilos (como fontes, cores e espaçamentos) a documentos web.
JavaScript
Linguagem de programação interpretada estruturada, de alto nível, que permite implementar itens complexos em páginas web.
HTTP
Protocolo de transferência que permite a troca de informações e dados na World Wide Web.
DNS
Sistema que traduz nomes de domínio legíveis por humanos em endereços IP numéricos.
